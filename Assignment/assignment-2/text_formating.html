Web technology refer to the way computers/devices communicate with each other using markup languages. It invo it is communication across the web, and create, deliver or manage web content using hypertext markup language(HTML).
A web page is a web document which is written in HTML(hypertext markup language).
It is said to have brought the world into a small village where people and devices can communicate to each other seamleslly.

Web Browser

A web browser is an application program for retrieveing, presenting and traversing information resources on the World Wide Web.
An information resource is identified by a uniform resource identifier (URL) and may be a web page, image, video or other piece of content.

Example:Internet Explorer, Google chrome, Firefox, Opera, Safari etc.

Web server

A special high end computer that hots a website on internet. Today we have cloud service that acts as a web servers. A web server is software and hardware that uses HTTP and others protocols to respond to clinet request made over the World Wide Web.The main job of web server is to display website content through storing, processing and delivering webpages to use.
